# Blank

`Blank`는 공백 문자를 나타내는 TypeScript 타입입니다.

:::tip
공백 문자(' ', '\n', '\t')를 표현하는 타입이 필요할 때 유용합니다.
:::

## 사용 예시

```ts
import { P } from 'utilscript';

type BlankType = P.Blank;
```
